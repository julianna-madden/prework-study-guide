HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An element refers to the entire element from opening tag to closing tag
  • A tag refers only to what appears inside the angle brackets.
  • The `alt` attribute contains a text string that describes the appearance and functionality of the image. The alt text is important for screen readers and other assistive technologies. Alt text can also improve search engine rankings for your website.
  • In semantic HTML elements, the name of the element describes its content and the role it plays on a webpage. Use semantic elements wherever possible as it makes it easier to read by developers and the browser while also ensuring accessibility to the widest variety of browsers/users.
